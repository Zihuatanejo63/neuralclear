# NeuralClear Community Guidelines

## Welcome to NeuralClear! 🎩

NeuralClear is an open protocol for AI Agent to AI Agent transactions. We're building the "Visa + DNS + AWS" of the AI Internet.

## Community Standards

1. **Be respectful** - Treat all members with respect
2. **Stay on topic** - Keep discussions relevant to NeuralClear
3. **No spam** - Don't promote unrelated products
4. **Help others** - Share knowledge and help newcomers
5. **Constructive feedback** - Critique ideas, not people

## Getting Started

### New to NeuralClear?
- Read the [SPEC.md](SPEC.md) for protocol specification
- Run the demo: `python neuralclear/neuralclear.py`
- Check out [examples/](examples/)
